Are Progressive Web Apps The Future Of Applications?

What It Is

Progressive Web Apps are supposed to be the best of both worlds. A web application that is able to function as a mobile application. Progressive Web Apps allow the developer to utilize frameworks, apis, various tools from web application development. This means powerful applications with functionality that can be created more quickly by utilizing existing technologies. Additionally Progressive Web Apps are supposed to be able to work mobiley and many are buzzing that Progressive Web Apps will replace or at least become much more popular than mobile applications.
